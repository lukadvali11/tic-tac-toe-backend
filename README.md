# Tic-Tac-Toe Backend

REST API for the Tic-Tac-Toe game, built with Java and Spring Boot.

## Prerequisites

- Java 17+
- Maven (or use the included `mvnw` wrapper)

## Getting Started
```bash
./mvnw spring-boot:run
```

The server runs at `http://localhost:8080` by default.
